ShortRouteMap.O

January 2016 – Present

Shortest round-trip planner website

Technologies: AngularJS, bootstrap, HTML5/CSS3, jQuery, php, Google Maps REST APIs, CouchDB

  • A round-trip organizing website that gives options of places one would want to visit on a trip and based on the selection gives shortest and the most convenient round-trip route on the Map.
  • Result window integrates Google Maps and displays the round-trip directions based on the places selection.
  • Uses geocoding and Google Maps Places api for location information, ratings and nearby places.
  • The round-trip algorithm ensures that all places are covered with the minimum travel distance and time.
  • The website also provides added features like share result, save searches and user specific suggestions(Future Scope).

Forecast Search Website

October 2015 – November 2015

Responsive design for Weather forecast

Technologies: HTML5/CSS3, Bootstrap, JavaScript, jQuery, PHP, JSON, Amazon EC2

  • A Weather Forecast Search Website that provides weather details for a requested city in united states.
  • The results section gives 3 tabs for current, next 24 hours and next 7 days weather forecast information (Default tab shows current details).
  • Website also integrates a map with the use of OpenWeatherMap OpenLayers.js library which helps to create the cloud map.
  • Facebook share functionality enables to share the result on facebook.
  • The website has been designed to be responsive to the screen sizes keeping "mobile first" concept in mind.
